# Outline Luminous Rail 文档目录设计 v2

**日期**: 2026-06-10  
**状态**: 已批准实施  
**取代**: Ghost Spine 文字列表方案

## 摘要

目录为编辑画布左缘的 **光轨（Luminous Rail）**：窄轨 + 层级刻度 + 单条浮动标题。无滚动条、无 12rem 侧栏、无「目录 N 项」面板头。

## 交互

- 收起：竖线 + ListTree 把手（~1.5rem）
- 展开：刻度轨（~1.75rem）+ 悬停/擦洗时单条 caption
- 滚轮在轨上：擦洗章节索引，不滚动列表
- `Ctrl/Cmd+Shift+O` / Esc / 把手切换

## 层级

H1/H2/H3 用刻度宽度与粗细区分，非文字缩进列表。

## 非目标

全文列表、虚拟滚动、折叠树、目录内搜索。
