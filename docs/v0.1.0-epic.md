## 验收标准（来自 ROADMAP）

- [ ] 用户可以在 Iris 中创建、编辑、删除笔记
- [ ] 用外部编辑器修改 `.md` 文件后，Iris 可以检测并提示同步
- [ ] 内联 AI 的接受/重试/回退操作正常工作
- [ ] 语义搜索的 Top-5 召回率达到可用水平（Recall@5 ≥ 0.6，见 `docs/eval/semantic-search.md`）
- [ ] API Key 不落盘明文，存储于操作系统凭据管理器
