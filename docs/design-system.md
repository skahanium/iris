## B · 纸墨编辑（主方向）

### 气质

长文写作优先：编辑区像**纸**，应用外壳像**墨**（低对比 chrome）。AI 是校对台与侧栏助手，不是聊天 App 主屏。

### 分区

| 区域 | 角色 | 默认观感 |
|------|------|----------|
| **Chrome** | 标签栏、状态栏、AI 侧栏外壳、对话框 | 中性炭灰（冷灰低饱和），非泛黄棕 |
| **Editor canvas** | TipTap 主编辑区 | 近白/浅灰纸栏、衬线正文、居中约 42rem，壳与纸面分层 |
| **Accent** | 链接、主按钮、AI 标识 | 克制赭铜（**不用** violet 紫） |

### 色彩 token（CSS 变量）

实现见 `src/styles/globals.css`。

| Token | 暗色 chrome 默认 | 用途 |
|-------|------------------|------|
| `--background` 等 | 冷灰炭墨 `hsl(240 6% 7%)` 系 | 外壳、侧栏 |
| `--editor-paper` | 近白冷纸 `hsl(40 4% 99%)`（亮）/ 浅灰纸 `hsl(40 6% 90%)`（暗壳上） | 居中纸栏 |
| `--editor-ink` | 中性墨 `hsl(240 9% 11%)` 系 | 正文 |
| `--primary` | 赭铜 `hsl(22 36% 42%)` 系 | 主操作、AI 强调 |

亮色主题（`.light`）：壳为中性浅灰，纸栏近白；暗色主题：壳为炭灰，纸栏为柔和浅灰纸，保持「纸比壳亮」的层次。

### 字体

| 场景 | 栈 | 说明 |
|------|-----|------|
| **编辑器正文** | `font-editor` | `"Noto Serif SC"`, `"Source Han Serif SC"`, `Georgia`, serif |
| **UI / 侧栏** | `font-sans` | 系统无衬线（苹方 / 微软雅黑） |
| **代码块** | `font-mono` | JetBrains Mono 等等宽 |

### 间距与栏宽

- 编辑区：`max-width: 42rem`，边距 `clamp` 响应式，行高 `1.65`，字间距 `0.012em`
- 侧栏：固定 `280px`（C 备选：可收起，见下）

### AI 组件（B）

- **引用卡**：纸色底 + 细赭石边，来源文件一行 meta，正文 `line-clamp`，操作弱化
- **对话泡**：避免 Messenger 式大块彩色；用户消息浅灰底，助手消息细边框
- **流式内联节点**（`ai-stream`）：与 accent 同系，勿用紫色渐变

## C · 命令优先（备选原则）

不单独做一套视觉，而是**同一套 B 的 token** 下调整布局与快捷键：

| 原则 | 现状 / 规划 |
|------|-------------|
| 导航 | `Ctrl+P` Quick Open 为主入口（已实现） |
| 搜索 | `Ctrl+Shift+F`（已实现） |
| 文件树 | `Ctrl+Shift+E` Sheet，非常驻（已实现） |
| **AI 侧栏** | `Ctrl+Shift+A` 收起/展开（已实现基础版） |
| 弱化常驻 chrome | 后续：标签栏自动隐藏、Zen 模式（未做） |
