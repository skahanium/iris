# 城市骑行基础设施的规划与设计

城市骑行基础设施的规划是现代都市向可持续交通转型的关键环节。荷兰和丹麦是公认的全球骑行友好城市典范，其成功的核心理念是将自行车视为独立的交通方式而非机动车交通的附属品。哥本哈根的"自行车高速公路"网络连接市区与郊区，确保通勤骑行者在至少 20 公里的路程上享有连贯的专用路权，交叉路口的信号灯为骑行者设置了"绿波"——以 20 公里每小时的速度骑行可连续通过多个绿灯。

骑行基础设施的分级体系决定了安全性和舒适度。第一级为完全物理隔离的自行车专用道（Protected Bike Lane），通过路缘石、绿化带或停车隔离带与机动车道分隔，是唯一能显著吸引新手骑行者上路的设计形式。第二级为彩色铺装的划线自行车道，虽在法律上界定为专用车道但无物理防护，停车或转弯的机动车侵入频率高。第三级为共享道路（Sharrow），仅施划地面标志提示机动车与自行车共享车道，安全保障作用微弱。

交叉口设计是骑行网络中最危险也最复杂的节点。受保护的交叉口（Protected Intersection）通过转角安全岛、专用信号相位和缩窄的转弯半径来实现机动车的低速右转和骑行者的视线通畅，将侧碰和右转钩形事故的风险降至最低。荷兰的"可持续安全"（Duurzaam Veilig）交通愿景将道路分为"通过功能"和"到达功能"两类，在通过型干道上完全分隔机动车与慢行交通，在到达型社区街道上通过物理减速措施营造机动车与自行车自然混合的低速共享空间。

公共自行车系统和微型出行工具的整合是近年来的热点。共享单车和无桩电动滑板车解决了"最后一公里"问题但如果缺乏停靠管理和路权规则就会迅速退化为街道混乱。有效的城市骑行治理需要三要素的同步推进：物理基础设施的持续投资、交通法规对弱势道路使用者的优先保护、以及通过骑行教育和文化推广改变市民的出行观念。
