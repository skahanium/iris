Iris AI Agent Upgrade Options

A visual comparison for turning the current chat panel into a manual research-and-writing workflow agent.

A

Workflow Agent Core

User picks a workflow, Iris runs a visible plan: local retrieval, optional web search, evidence ranking, citation checking, then document patch preview.

B

Chat Plus Tools

Keep the chat panel as the main surface, add better search tools, source cards, and confirmable insert/replace actions.

C

Autonomous Agent Mode

User gives a goal, Iris plans and loops through tools more freely. Powerful, but harder to evaluate and riskier for Markdown writes.

Why A fits Iris

  • Feels like a specialized notes agent, not a generic chatbot.
  • Search quality becomes observable and testable.
  • Markdown writes stay confirmable and reversible.

Design cost

  • Needs a workflow state model, not only message history.
  • Requires web/local evidence to share one packet interface.
  • Needs stronger patch and file-title rules.
